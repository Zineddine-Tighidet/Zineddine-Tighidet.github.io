## Deploy to GitHub Pages

1. Create a repository named `USERNAME.github.io` on GitHub, replacing `USERNAME` with your GitHub username.
2. Push this folder's contents to that repository's default branch (usually `main`).

```bash
cd "$(dirname "$0")"
git init
git branch -M main
git remote add origin https://github.com/USERNAME/USERNAME.github.io.git
git add .
git commit -m "Initial personal site"
git push -u origin main
```

3. Visit `https://USERNAME.github.io` after a minute or two.

To customize:
- Update text in `index.html`, `about.html`, `projects.html`, `contact.html`.
- Add your email to the `mailto:` link in `contact.html`.
- Tweak styles in `assets/style.css`.
